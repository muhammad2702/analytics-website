# Data Checks

A small Python project for checking tabular data quality.

## What it checks

- Missing values
- Duplicate rows
- Invalid numeric ranges
- Invalid category values
- Basic summary statistics

## Files

- `data_checks.py` - validation script
- `sample_data.csv` - sample input data

## Run

```bash
python data_checks.py
```

## Resume line

Built a Python data validation script that checks missing values, duplicate rows, invalid ranges, category errors, and summary statistics for tabular datasets.
